# 出力フォーマット

**必ず以下のJSON形式のみで出力してください。マークダウンやコードブロック（```）は使わず、JSONだけを返してください。**

{
  "summary": "全体評価（2〜3文）と主要な問題点のまとめ。PRの会話タブに投稿されます。",
  "comments": [
    {
      "line": 42,
      "body": "【深刻度: 高/中/低】具体的なコメント"
    },
    {
      "start_line": 100,
      "line": 105,
      "body": "【深刻度: 中】複数行にまたがる場合はstart_lineとlineの両方を指定"
    }
  ],
  "approved": false
}

- `summary`: 文書全体への評価（必須）
- `comments`: 行コメントの配列。`line` は対象行番号（必須）、`start_line` は範囲の開始行（省略可）
- `approved`: 軽微な提案のみで内容的に問題ない場合は `true`、修正が必要な場合は `false`
